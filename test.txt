Here's a simple Flask application that allows users to upload and download files to/from an S3 bucket. Make sure you have the `boto3` library installed (`pip install boto3`) and set up your AWS credentials properly.

```python
from flask import Flask, request, render_template, send_file
import boto3
import os

app = Flask(__name__)

# Configure your AWS credentials and S3 bucket name
AWS_ACCESS_KEY_ID = 'your_access_key'
AWS_SECRET_ACCESS_KEY = 'your_secret_key'
S3_BUCKET_NAME = 'your_bucket_name'

# Initialize the S3 client
s3_client = boto3.client(
    's3',
    aws_access_key_id=AWS_ACCESS_KEY_ID,
    aws_secret_access_key=AWS_SECRET_ACCESS_KEY
)

@app.route('/')
def index():
    return render_template('index.html')

@app.route('/upload', methods=['POST'])
def upload():
    if 'file' not in request.files:
        return 'No file part'
    file = request.files['file']
    if file.filename == '':
        return 'No selected file'
    if file:
        s3_client.upload_fileobj(file, S3_BUCKET_NAME, file.filename)
        return 'File uploaded successfully'

@app.route('/download', methods=['POST'])
def download():
    file_key = request.form['file_key']
    if not file_key:
        return 'No file key provided'
    local_filename = os.path.join('/tmp', file_key)
    s3_client.download_file(S3_BUCKET_NAME, file_key, local_filename)
    return send_file(local_filename, as_attachment=True)

if __name__ == '__main__':
    app.run(debug=True)
```

Create an `index.html` file in a `templates` folder with the following content:

```html
<!doctype html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">
    <title>Upload and Download to S3</title>
  </head>
  <body>
    <div>
      <h1>Upload File to S3</h1>
      <form action="/upload" method="post" enctype="multipart/form-data">
        <input type="file" name="file">
        <input type="submit" value="Upload">
      </form>
    </div>
    <div>
      <h1>Download File from S3</h1>
      <form action="/download" method="post">
        <input type="text" name="file_key" placeholder="Enter the S3 file key">
        <input type="submit" value="Download">
      </form>
    </div>
  </body>
</html>
```

Make sure to replace `'your_access_key'`, `'your_secret_key'`, and `'your_bucket_name'` with your actual AWS credentials and S3 bucket name. This application allows users to upload files to S3 and download files from S3 using simple HTML forms.
